# stream-AI

Minimal Next.js + Tailwind starter for a dark-theme streaming chat UI.

Important: DO NOT commit any API keys. Add your Gemini/OpenAI key in your host or a local `.env.local` file (ignored by git).

Quick start:
1. Install: `npm install`
2. Add your key locally (create `.env.local`):
   `GEMINI_API_KEY=your_key_here`
3. Run dev server: `npm run dev`
4. Open `http://localhost:3000`

Deployment: set `GEMINI_API_KEY` (or your provider key) in your host's environment variables (Vercel, Render, Cloud Run).
